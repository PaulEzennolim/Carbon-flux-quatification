  Universal-TEMPO : Fine-tuned on all 5 training sites (FI-Lom, GL-ZaF,
                    IE-Cra, DE-Akm, FR-LGt) — all classified as wetlands.
  Wetland-TEMPO   : Same data as Universal. Explicit wetland config to test
                    H1 and document the implicit wetland training bias.
  Forest-TEMPO    : Adapted from Universal checkpoint using first 70% of
                    SE-Htm (temporal split). Evaluated on remaining 30%.
  Zero-Shot-TEMPO : Pretrained TEMPO-80M, no fine-tuning (baseline).

  KEY INSIGHT: ALL training sites are wetlands. Universal-TEMPO therefore
  has an implicit wetland bias. Forest adaptation corrects this via
  in-domain temporal fine-tuning on SE-Htm.

  METHODOLOGICAL NOTE: All models are evaluated on identical test sets.
  For SE-Htm, Universal and Wetland predictions are sliced to the last
  30% (test split) so RMSE/MAE are directly comparable to
  Forest-TEMPO. Zero-Shot predictions were generated on a random 500-
  sample subset (no stored indices); they are aligned to the test split
  by length and serve as a reference baseline only.

NOVELTY STATEMENT

  First systematic evaluation of ecosystem-specific conditioning for a
  time-series foundation model (TEMPO) in carbon flux forecasting.
  Contributions:
  1. Demonstrates that universal training on biased ecosystem distributions
     (all-wetland) limits foundation model performance on forest sites.
  2. Quantifies cross-ecosystem transfer degradation (wetland→forest).
  3. Shows in-domain temporal adaptation as a practical remedy.
  4. Establishes a reusable framework for ecosystem-aware foundation
     model deployment in heterogeneous carbon monitoring networks.
